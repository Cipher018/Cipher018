## Naming Convention

> All systems, modules, firmware, and documents follow a unified naming standard.

| Type | Format | Example |
|---|---|---|
| Aerial platforms | `XDF-[NNN]` | `XDF-001`, `XDF-002` |
| Software modules | `[FUNCTION]-v[X.Y]` | `C2-v2`, `DATALINK-v2` |
| Firmware versions | `fw-[YYYYMMDD]` | `fw-20250424` |
| Payloads | `POD-[FUNCTION]` | `POD-SURV`, `POD-CARGO` |
| VTOL variants | `XDF-[NNN]-V` | `XDF-002-V` |
| Internal documents | `CI-DOC-[YYYYMMDD]-[TOPIC]` | `CI-DOC-20250424-ARCH` |
